Download ACEMD platform





ACEMD platform includes: HTMD, Moleculekit, Parameterize and ACEMD. You can download and use this platform for free, but it comes with a limit of 1 GPU.

If you want to use more GPUs, please contact us to get a quote for a license. We are flexible!

One more thing before installing, please read the EULA (End User License Agreement) for ACEMD platform.

Let's install it:

            #create a clean conda environment, with python 3.6
            conda create -n acellera_tools python=3.6 

            #activate the environment
            conda activate acellera_tools

            #HTMD, Moleculekit, Parameterize and ACEMD will be installed inside acellera_tools environment
            conda install -c acellera -c psi4 -c conda-forge acemd-platform
