Do you already have a license?

If you have a license, you only need to install HTMD following the guide below and then follow the instructions in the handling licenses section to activate it.

Getting HTMD

Getting the package installer

First, check if you already have conda:

conda -V

If this command reports the conda version, follow to the Installing HTMD section. If not, download the appropriate following package:

Mac OS X
(Some features are not supported)

Linux

Windows
(Some features are not supported)

64-bit (bash installer)

64-bit (bash installer)

64-bit (exe installer)

Now, to install the conda package manager, go to directory where you downloaded the previous package and run:

bash Miniconda3-latest-*-x86_64.sh

And follow the instructions.

Installing HTMD

With conda installed, you can now easily install HTMD by running the following commands:

conda config --add channels acellera
conda config --add channels psi4
conda install htmd

Register HTMD

Before using HTMD, please read the EULA and register using the command

htmd_register

If you don't accept the EULA and register at this point, HTMD will prompt for registration when the Python module is imported

Handling your license.

If you have already purchased a license and install HTMD, you only have to activate the license:

If you have purchased an HTMD license from Acellera you will be issued with an activation token. Once the installation has finished, issue the command: 

 $ activate_license [activation-token]

If you have obtained HTMD Pro as part of a Metrocubo solution, you will receive the machine with ACEMD and HTMD installed, configured and ready to run.

Make sure you have added your installation path to the PATH environment variable. Also, sometimes closing the terminal you have used to do the installation and open a new one helps.

Start using HTMD

Start using HTMD by either starting an interactive Python shell:

ipython

Or by opening an ipython notebook:

jupyter-notebook

On either case, import the module and start using HTMD:

from htmd.ui import *

Updating HTMD

If you want to update HTMD to the latest version, run:

conda update -c acellera -c psi4 htmd

If you have any doubts, please do not hesitate to email us at support@acellera.com
