System Requirements
==================

* OS: CentOS >=6 (x86_64)
* CPU: x86_64 >1 GHz
* RAM: >8 GB
* HDD: >10 GB

* GPU: NVIDIA
* GPU RAM: >2GB
* NVIDIA driver >= 410.48

* Supported queues: Slurm, LSF, PBS

Altough some pieces of our software do not explicitely require a GPU, like Moleculekit, you will find it necessary to run MD simulations.
